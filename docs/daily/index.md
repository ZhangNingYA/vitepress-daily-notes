# 🌿 每日入口


<div class="quote-text">
“这里不是打卡机，是你给自己留的一束光。<br>
今天也一样：把学习拆小一点，把坚持做轻一点。”
</div>

::: tip 💌 给你的小纸条
嗨，我的🐖。

今天不需要“变得很厉害”，只要比昨天多留下一个小小的痕迹就好。  
这里记下来的每一行，都会在某个未来的回看里，悄悄变成你的底气。  

**慢慢来，你一直在走。**
:::

## 🍵 今日仪式感

<div class="daily-cards">

<div class="card">

#### 1. 拾贝壳 (单词句型)
*选 3-5 个你真的会用的*
- **积累**：写下词汇/短语
- **造句**：1 个真实语境句型
- **思考**：我明天要怎么用它？

</div>

<div class="card">

#### 2. 磨工具 (语法)
*只学一个点，不贪多*
- **规则**：规则 + 1 个例句
- **陷阱**：写 1 个错误反例
- **复盘**：我最容易错在哪里？

</div>

<div class="card">

#### 3. 碎碎念 (心得)
*三句话也可以*
1. 我今天做到了什么
2. 我卡在了哪里
3. 明天怎么更轻松一点

</div>

</div>

## 📝 今日小目标

> *你可以很“轻”地完成，打钩的那一刻最解压。*

- [ ] **拾贝壳**：3 个单词 + 1 个句型
- [ ] **磨工具**：1 个语法点 + 2 句例句
- [ ] **碎碎念**：写下 3 句真心话
- [ ] **晚安礼**：睡前把今天最喜欢的一句标红/加粗

::: details 🍃 如果今天状态一般（点这里）
**那就做“最小版本”，依然值得骄傲：**

- 单词：写 **1 个** 就好
- 语法：看 **1 条** 就好
- 心得：写 **1 句** 哪怕是“今天好累”

只要留下痕迹，今天就算赢。
:::

## 🕰️ 时间胶囊

| 频率 | 动作 | 意义 |
| :--- | :--- | :--- |
| **Daily** | 睡前 1 分钟扫视 | 像鱼一样在记忆里游一圈 |
| **Weekly** | 挑选 3 个“本周最爱” | 所有的努力都值得被加精 |
| **Monthly** | 归档里回看 10 条 | 看，你已经把自己养得这么好了 |
